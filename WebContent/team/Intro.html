HTML5 Audio/Video Methods

Method Description
addTextTrack() (지원되지 않음)
canPlayType() 현재 브라우저가 특정 오디오/비디오 포맷을 재생할 수 있는지 확인
load() 오디오/비디오 element 다시 불러오기
play() 오디오/비디오 재생
pause() 현재 재생 중인 오디오/비디오 일시정지

HTML5 Audio/Video Properties

Property Description
audioTracks (지원되지 않음)
autoplay 오디오/비디오가 불러지고 바로 재생하는지 여부를 설정
buffered 오디오/비디오의 버퍼 구간(지정한 시작~끝 지점) 반환
controller (지원되지 않음)
controls 오디오/비디오의 기본 컨트롤러 표시 여부
crossOrigin (사용 예시 없음)
currentSrc 현재 오디오/비디오의 URL주소를 반환
currentTime 현재 재생 시간을 반환하거나 설정
defaultMuted 기본값으로 음소거 설정
defaultPlaybackRate 오디오/비디오의 기본 재생 속도 확인 또는 설정
duration 현재 오디오/비디오의 재생시간 확인
ended 오디오/비디오의 재생이 끝났는지 확인
error (지원되지 않음)
loop 오디오/비디오의 반복 재생 설정
mediaGroup 같은 아이디로 묶어준 비디오/오디오들 그룹화
muted 음소거 설정
networkState 현재 오디오/비디오의 네트워크 상태 반환
paused 오디오/비디오의 일시정지 상태 확인
playbackRate 오디오/비디오의 재생 속도 확인 또는 설정
played 오디오/비디오의 재생된 구간 반환
preload 페이지 로드 후 오디오/비디오가 로드될지 설정
readyState 오디오/비디오의 준비 상태 반환
seekable 탐색 가능 영역의 시간대 반환
seeking 오디오/비디오의 탐색 여부 확인
src 현재 오디오/비디오 element의 소스 설정하거나 리턴
startDate (지원되지 않음)
textTracks 이용 가능한 가사/자막 목록 보여주기
videoTracks (지원되지 않음)
volume 오디오/비디오의 음량 설정
