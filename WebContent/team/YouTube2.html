1번 째 영상(Autoplay)

영상이 자동적으로 실행되는곳
실행 구문: src = "주소?autoplay=1"

2번째 영상(recommended)

기본적인 iframe 을 써서 동영상 재생
실행 구문 : src = "https://www.youtube.com/embed/erErBFKPbMY"

3번째 영상(Loop)

영상이 끝나고 난뒤 다시 재생 되도록 설정 한곳.
실행 구문: src = "주소?playlist=erErBFKPbMY&loop=1"

4번째 영상(Controls)

밑에 있는 재생버튼 안보이도록 하는 구문.
실행 구문: src = "주소?controls=0"
